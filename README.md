# Nonparametric-Tests-for-Recurrent-and-Terminal-Events-with-Physiological-Decline

estimation.R contains the core functions used to estimate the area under the longitudinal profile of τ-restricted, utility-adjusted mean event-free times for group g, g = 1, 2; and the variances, as described in the manuscript.

helper.cpp contains the helper functions used in estimation.R, written in C++.

Data.R contains the code used to simulate the datasets described in the simulation section of the manuscript.

Azith.csv is the dataset used for the example section of the manuscript (The Azithromycin for Prevention of COPD Exacerbations Trial).
